# Titan Habitability Pipeline - Read Me

This code base represents a pipeline for computing the habitability of the moon Titan from the past into the present and 
onto the future. It was written as part of a thesis for the MPhil in Planetary sciences and Life in Universe at 
Cambridge University 2025-2026.

Please read the installation instructions in the file [INSTALL.md](INSTALL.md) and the outstanding issues document at [ISSUES.md](ISSUES.md)
The entry point to the pipeline is [run_pipeline.py](run_pipeline.py). It is heavily documented.

The report write up/thesis is not presented here. Please contact me for a copy ([cm10004@cam.ac.uk](mailto:cm10004@cam.ac.uk)).

Although this project refers to many research papers, they are not presented here for copyright reasons.
